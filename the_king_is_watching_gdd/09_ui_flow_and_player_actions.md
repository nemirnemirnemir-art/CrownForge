# 09. UI-потоки и действия игрока (UI Flow & Player Actions)

## Обзор интерфейса

Игра использует минималистичный интерфейс, сфокусированный на сетке замка 5×5 и управлении взглядом короля.

---

## Основные экраны

### 1. Главное меню / Хаб (Town Hub)

```
┌─────────────────────────────────────────────────┐
│                   ХАБ ГОРОДА                     │
├─────────────────────────────────────────────────┤
│                                                  │
│  [НАЧАТЬ ЗАБЕГ]     [УЛУЧШЕНИЯ]                  │
│                                                  │
│  [СОВЕТНИКИ]        [КОРОЛИ]                     │
│                                                  │
│  [ЧЕЛЛЕНДЖИ]        [НАСТРОЙКИ]                  │
│                                                  │
└─────────────────────────────────────────────────┘
```

| Элемент | Функция |
|---------|---------|
| **Начать забег** | Переход к выбору карты/Threat Level |
| **Улучшения** | Мета-прогрессия, трата ресурсов |
| **Советники** | Выбор советников для забега |
| **Короли** | Разблокировка и выбор королей |
| **Челленджи** | Просмотр и получение наград за челленджи |

## Действия игрока

### Управление взглядом

| Действие | Управление |
|----------|------------|
| **Перемещение взгляда** | Перетаскивание мышью |
| **Вращение взгляда** | Правый клик (ПКМ) |
| **Апгрейд взгляда** | Клик на "+" рядом с иконкой глаза |

### Управление зданиями

| Действие | Управление |
|----------|------------|
| **Размещение здания** | Выбор чертежа → клик на пустую клетку |
| **Перемещение здания** | Shift + ЛКМ на здании → перетаскивание |
| **Просмотр информации** | Наведение на здание |

### Управление армией

| Действие | Управление |
|----------|------------|
| **Авто-спавн** | Переключатель "Spawn Troops to Capacity" |
| **Ручной деплой** | Unknown (возможно кнопка или автоматически) |

### Управление заклинаниями

| Действие | Управление |
|----------|------------|
| **Выбор заклинания** | Клик на иконку заклинания |
| **Применение** | Клик на цель / область |

### Взаимодействие с NPC

| Действие | Управление |
|----------|------------|
| **Торговец** | Клик на иконку/кнопку торговца |
| **Оракул** | Автоматически между ключевыми волнами |

### Скорость игры

| Режим | Условие |
|-------|---------|
| **Обычная скорость** | По умолчанию |
| **Двойная скорость** | После разблокировки мета-улучшения |

## Потоки действий

### Поток: Начало забега

```
ХАБ → Выбор карты → Выбор Threat Level → Выбор короля 
    → Выбор советников → СТАРТ
```

### Поток: Игровой цикл

```
┌──────────────────────────────────────────────────────┐
│                                                       │
│  ┌─────────────┐                                      │
│  │ ПОДГОТОВКА  │ ← Размещение зданий                  │
│  │             │ ← Управление взглядом                │
│  │             │ ← Производство ресурсов              │
│  └──────┬──────┘                                      │
│         ▼                                             │
│  ┌─────────────┐                                      │
│  │   ВОЛНА     │ ← Бой с врагами                      │
│  │             │ ← Использование заклинаний           │
│  │             │ ← Управление взглядом на казармы     │
│  └──────┬──────┘                                      │
│         ▼                                             │
│  ┌─────────────┐                                      │
│  │  НАГРАДЫ    │ ← Выбор артефакта                    │
│  │             │ ← Посещение торговца                 │
│  │             │ ← Событие (случайно)                 │
│  └──────┬──────┘                                      │
│         │                                             │
│         ▼                                             │
│     [БОСС?] ──да──► Оракул → Волна босса              │
│         │                                             │
│        нет                                            │
│         │                                             │
│         └───────────────────────────┘                 │
│                                                       │
└──────────────────────────────────────────────────────┘
```

### Поток: Конец забега

```
HP замка = 0  ──────► ПОРАЖЕНИЕ ──┐
                                  │
3 босса побеждены ──► ПОБЕДА ─────┤
```

## Элементы HUD

### Верхняя панель

| Элемент | Информация |
|---------|------------|
| **Ресурсы** | Текущее количество каждого ресурса |
| **Мораль** | Уровень морали (5 звёзд) |
| **Волна** | Номер текущей волны |

### Нижняя панель

| Элемент | Информация |
|---------|------------|
| **Чертежи** | Доступные для размещения здания |
| **Апгрейд взгляда** | Кнопка + стоимость |
| **Заклинания** | Доступные заклинания (6 слотов) |
| **Армия** | Текущее/максимальное количество юнитов |
| **HP замка** | Полоса здоровья замка |

### Индикаторы зданий

| Элемент | Значение |
|---------|----------|
| **Колесо прогресса** | Прогресс производства |
| **Шеврон** | Количество апгрейдов здания |
| **Подсветка** | Под взглядом / вне взгляда |
