# 00. Обзор игры "The King is Watching"

## Краткое описание (Pitch)

**The King is Watching** — это roguelite-стратегия с элементами строительства королевства, разработанная студией **Hypnohead** и изданная **tinyBuild**. Уникальная механика игры заключается в том, что все здания и производство в королевстве работают **только под взглядом короля**. Игрок управляет ограниченной зоной внимания короля (гейзом), решая, какие здания активировать в каждый момент времени, балансируя между добычей ресурсов, тренировкой войск и обороной замка.

**Дата выхода:** 21 июля 2025  
**Платформа:** Windows (Steam)  
**Цена:** $15  
**Жанр:** Roguelite / Kingdom Builder / Tower Defense

---

## Ключевая игровая фантазия

Игрок — **всевидящий король**, чей взгляд буквально оживляет королевство. Подданные работают только когда король смотрит на них; стоит отвернуться — и они бездельничают. Это создаёт постоянное напряжение: нужно решать, куда направить внимание — на поля для еды, шахты для богатства или казармы для обороны.

---

## Ключевой дифференциатор: механика взгляда (Gaze)

Центральная механика, отличающая игру от конкурентов:

| Аспект | Описание |
|--------|----------|
| **Зона взгляда** | Тетрис-образная фигура из клеток, которую можно перемещать и вращать |
| **Начальный размер** | 3 клетки |
| **Максимальный размер** | До 6 клеток (апгрейд за ресурсы) |
| **Эффект** | Только здания под взглядом работают и производят ресурсы/юнитов |
| **Вне взгляда** | Здания останавливают производство |

## Основные системы игры

| Система | Описание |
|---------|----------|
| **Взгляд (Gaze)** | Управление зоной активации зданий |
| **Ресурсы** | 14 типов: базовые и продвинутые |
| **Здания** | Производство, казармы, улучшения, утилиты |
| **Юниты** | Ближний бой, дальний бой, маги, осадные |
| **Мораль** | Влияет на характеристики войск |
| **Заклинания** | Активные способности для боя |
| **Артефакты** | Пассивные и активируемые бонусы |
| **Уровень угрозы** | Масштабируемая сложность (Threat Level) |
