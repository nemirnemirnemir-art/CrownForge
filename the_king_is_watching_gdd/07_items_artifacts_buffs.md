# 07. Предметы, артефакты и баффы (Items, Artifacts & Buffs)

## Обзор системы артефактов

**Артефакты** — это предметы, дающие пассивные или активируемые бонусы во время забега. Они влияют на войска, здания, ресурсы и механики боя.

---

## Способы получения артефактов

| Источник | Описание |
|----------|----------|
| **Награды за волны** | Выбор из 3 предложенных |
| **Торговец** | Покупка за золото |
| **Крипты (Graveyard)** | Обмен реликвий на артефакты |
| **События** | Случайные награды |
| **Advisor бонусы** | Стартовые артефакты от советников |

---

## Механика выбора

- После волн предлагается **выбор из 3 артефактов**
- Артефакт "Voodoo Beads" делает первый реролл бесплатным
- Можно отказаться от выбора (Unknown: подтвердить)

## Реликвии (Relics) — только Graveyard

### Что такое реликвии?

**Реликвии** — это особые предметы, которые можно найти только на карте **Graveyard** в криптах на поле боя.

### Механика реликвий

| Аспект | Описание |
|--------|----------|
| **Источник** | Крипты (Crypts) на поле боя |
| **Использование** | Обмен на войска, улучшения, товары |
| **Особенность** | Уникальны для карты Graveyard |

### Типы обмена реликвий

| Обмен | Результат |
|-------|-----------|
| **На войска** | Дополнительные юниты |
| **На улучшения** | Апгрейды зданий/юнитов |
| **На товары** | Ресурсы или артефакты |

## Система заклинаний (Spells)

### Обзор

Заклинания — это активные способности, которые игрок использует вручную во время боя.

### Получение заклинаний

| Источник | Описание |
|----------|----------|
| **Magic School** | Медленная генерация |
| **Magic College** | Быстрая генерация |
| **Награды** | Между волнами |
| **Артефакты** | Ancient Tome (+4 случайных) |

### Лимит заклинаний

| Параметр | Значение |
|----------|----------|
| **Слоты заклинаний** | 6 (по данным TheGamer) |
| **Разблокировка слотов** | Через мета-улучшения |

### Каталог заклинаний

По информации из стратегий сообщества:

| Заклинание | Тип | Эффект |
|------------|-----|--------|
| **Freeze** | Контроль | Замораживает врагов |
| **Sheep** | Контроль | Превращает врага в овцу |
| **Wrath** | Урон | Массовый урон |
| **Invulnerability** | Защита | Неуязвимость войск |
| **Heal Puddle** | Лечение | Область лечения |
| **Incineration** | Урон | Огненный урон |
| **Meteor** | Урон | Метеорит (малый) |
| **Fireworks** | Урон | Фейерверки (от артефакта) |

**Примечание:** Полный список заклинаний требует проверки wiki.

### Рекомендуемые заклинания против боссов

По стратегиям сообщества:
1. **Freeze** — обязательно против сильных боссов
2. **Sheep** — контроль одиночных целей
3. **Wrath** — массовый урон
4. **Invulnerability** — спасение армии в критический момент

## Баффы от королей

Каждый король предоставляет **уникальные баффы**:

### Пассивные способности (Passive Abilities)

Разблокируются через мета-прогрессию (King's Quest):

| Уровень | Разблокировка |
|---------|---------------|
| **King's Quest 1** | Первая пассивка для всех королей |
| **King's Quest 2** | Вторая пассивка |
| **King's Quest 3** | Третья пассивка |

### Активные способности (Active Abilities)

Разблокируются выполнением челленджей (обычно победа над несколькими боссами):

| Король | Ресурсы для способностей |
|--------|--------------------------|
| **Baldwin** | Gold, Water |
| **Alucard** | Flour, Gold, Water |
| **Brezhnius** | Crystal, Water |
| **Leo** | Wheat, Wood |
| **Leonid** | Meat, Wine |
| **Saladin** | Water, Wheat, Meat |
| **Spellus** | Water, Oil, Gold |

## Баффы морали

### Эффекты морали на характеристики

| Уровень морали | Эффект (примерно) |
|----------------|-------------------|
| **Very Bad** | -% к характеристикам |
| **Bad** | Небольшие штрафы |
| **Neutral** | Без эффектов |
| **Good** | +% к характеристикам |
| **Very Good** | Максимальные бонусы |

### Эффекты морали по королям

Каждый король имеет уникальный "Morale Effect":

| Король | Бонус морали (пример) |
|--------|----------------------|
| **Baldwin** | Увеличение урона юнитов |
| **Leonid** | Unknown (связано с войсками) |
