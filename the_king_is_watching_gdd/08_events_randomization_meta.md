# 08. События, рандомизация и мета-прогрессия

## Система событий (Events)

### Обзор

**События** — это случайные ситуации, возникающие во время забега, требующие от игрока принятия решения.

### Триггеры событий

| Триггер | Описание |
|---------|----------|
| **Между волнами** | После завершения волны |
| **Случайно** | В процессе игры |
| **Специальные условия** | Unknown (требует проверки) |

### Примеры событий

По информации из обзоров:

| Событие | Описание |
|---------|----------|
| **Ленивый крестьянин** | Обнаружен бездельник, выбор последствий |
| **Растущий культ** | Секта в королевстве, нужно решить проблему |
| **Другие** | Различные нарративные ситуации |

### Структура выбора

Каждое событие предлагает **несколько вариантов** действий:

| Аспект | Описание |
|--------|----------|
| **Варианты** | 2-3 опции |
| **Последствия** | Ресурсы, мораль, войска, артефакты |
| **Риск** | Некоторые выборы имеют негативные последствия |

### События карты Volcano

По данным обновления:
- **20 новых уникальных нарративных событий**
- Тематически связаны с демонами и инферно

## Система рандомизации (RNG)

### Пулы случайностей

| Пул | Что генерируется |
|-----|------------------|
| **Чертежи зданий** | Какие здания предлагаются игроку |
| **Артефакты** | Выбор из 3 при награде |
| **Боссы** | 3 случайных босса за забег |
| **Враги волн** | Состав вражеской волны |
| **События** | Какое событие произойдёт |
| **Генералы** | Какой генерал командует волной |

### Разбавление пула (Pool Dilution)

По обсуждениям сообщества:

> "Возможно, стоит избегать разблокировки дополнительных чертежей — они разбавляют пул карт и уменьшают шанс увидеть нужную карту"

| Механика | Описание |
|----------|----------|
| **Разблокировка зданий** | Добавляет их в пул |
| **Эффект** | Меньше шанс получить конкретное здание |
| **Стратегия** | Осторожно разблокировать новые здания |

### Выбор волн (Wave Selection)

При посещении **Оракула** игрок может влиять на:

| Выбор | Влияние |
|-------|---------|
| **Порядок врагов** | Какие враги идут первыми |
| **Интенсивность** | Сила волн |
| **Награды** | Какие награды за победу |

## Мета-прогрессия (Meta Progression)

### Обзор системы

После каждого забега (победа или поражение) игрок получает ресурсы для **перманентных улучшений**:

| Ресурс | Источник |
|--------|----------|
| **Town Denarii** | Победа над волнами и боссами |
| **Ascension Crystals** | Победа на новом Threat Level, выполнение челленджей |

### Деревья улучшений

В игре **4 ветки улучшений**:

| Цвет | Фокус |
|------|-------|
| **Красная** | Боевые улучшения |
| **Жёлтая** | Экономика и ресурсы |
| **Синяя** | Советники и короли |
| **Зелёная** | Unknown / Смешанное |

### Ключевые мета-улучшения

#### Размер взгляда (Vision Size)

| Приоритет | Очень высокий |
|-----------|---------------|
| **Эффект** | +1 клетка к зоне взгляда |
| **Рекомендация** | Улучшать в первую очередь |

#### Вместимость армии (Unit Cap)

| Приоритет | Высокий |
|-----------|---------|
| **Эффект** | Больше юнитов в армии |
| **Рекомендация** | Критично для поздних волн и боссов |

#### Характеристики мечников (Swordsman Stats)

| Приоритет | Высокий |
|-----------|---------|
| **Эффект** | +HP, +урон мечников |
| **Рекомендация** | Укрепляет переднюю линию |

#### Двойная скорость игры (Double Game Speed)

| Приоритет | Средний |
|-----------|---------|
| **Эффект** | Возможность ускорить игру |
| **Рекомендация** | QoL улучшение |

#### Слоты заклинаний (Spell Slots)

| Приоритет | Средний |
|-----------|---------|
| **Эффект** | Больше заклинаний в запасе |
| **Рекомендация** | Зависит от стиля игры |

### King's Quest (Квесты короля)

Особая ветка улучшений для разблокировки пассивных способностей королей:

| Уровень | Эффект |
|---------|--------|
| **King's Quest 1** | Первая пассивка для ВСЕХ королей |
| **King's Quest 2** | Вторая пассивка для всех |
| **King's Quest 3** | Третья пассивка для всех |

## Система советников (Council/Advisors)

### Обзор

**Советники** — это персонажи, дающие стартовые бонусы перед забегом.

### Разблокировка

| Аспект | Описание |
|--------|----------|
| **Стартовый советник** | Gustav (разблокирован с начала) |
| **Всего советников** | 28+ (без Gustav) |
| **Слоты советников** | Разблокируются через синие улучшения |
| **Стоимость слотов** | Прогрессивно растёт |

### Механика слотов

| Слот | Разблокировка |
|------|---------------|
| **1-й** | С начала игры |
| **2-й** | Синие улучшения |
| **3-й** | Синие улучшения (дороже) |
| **4-й+** | Unknown |

### Примеры советников

| Советник | Бонус |
|----------|-------|
| **Gustav** | Стартовый (базовые бонусы) |
| **Automaton** | +5 Tree blueprints, +5 Wheat Field blueprints |
| **Amber Advisor** | Предоставляет мушкетёров |
| **Другие** | Различные бонусы |

## Уровни угрозы (Threat Levels)

### Механика

| Аспект | Описание |
|--------|----------|
| **Диапазон** | 1-10 |
| **Разблокировка** | Победа на предыдущем уровне |
| **Эффект** | Модификаторы сложности |

### Прогрессия

| Threat | Разблокировка | Награда |
|--------|---------------|---------|
| **1** | С начала | Базовая игра |
| **2** | Победа T1 | Unknown |
| **3** | Победа T2 | Карта Graveyard |
| **4-10** | Последовательно | Увеличенная сложность |

### Модификаторы

По данным wiki:
- Увеличение HP врагов
- Увеличение урона врагов
- Дополнительные типы врагов
- Изменение механик боссов
- Unknown: точные числа

## Рекомендуемый порядок мета-улучшений

По рекомендациям сообщества:

### Ранняя прогрессия

1. **Vision Size** — приоритет #1
2. **Unit Cap** — больше армии
3. **Swordsman Stats** — надёжная передняя линия
4. **Double Game Speed** — QoL

### Средняя прогрессия

1. **Council slots** — больше советников
2. **Spell slots** — больше заклинаний
3. **King's Quest** — пассивки королей
4. **Новые короли** — разнообразие стилей

### Поздняя прогрессия

1. **Специализированные улучшения** — по вашему стилю игры
2. **Разблокировка зданий** — осторожно (разбавление пула!)
3. **Высокие Threat Levels** — для челленджа
