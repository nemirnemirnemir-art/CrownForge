## Royal Decrees

Decrees are permanent run-wide buffs selected at Year 4 Royal Council event (and via Longtermism decree at future councils).

### Universal Basic Decrees (18 total)

| Decree | Effect | Max Stacks |
|--------|--------|-----------|
| Bunker | +50% tower attack speed | x20 |
| Feast | Doubles HP | x20 |
| Sharp Blades | +30% attack | x20 |
| Eyeglass | +20% crit chance | x5 |
| Iron Fist | Doubles crit damage | x3 |
| Leatherworks | Doubles movement speed | x20 |
| Poison Thorns | Enemies take poison on hit | x20 |
| Venom | Poison applied twice | x3 |
| War Horns | Summons 50% stronger | x20 |
| Catharsis | All troops deal damage equal to the highest-damage troop | x1 |
| Overwork | Support buildings apply effects one more time | x2 |
| Gunpowder | +30% AoE range | x5 |
| Loan | Gain 99 gold | x20 |
| Outsourcing | 9 free rerolls | x20 |

| Supplies | Receive a copy of all your king's cards | x20 |
| Weakspot | +50% damage vs frozen / stunned / poisoned enemies | x1 (implied) |

```
Royal Council (Yr 4)
  └─ Decree chosen → permanent run buff
  └─ Longtermism → 3 decrees at next council

Prophet (Yr 6)
  └─ Blessing foreseen → activates Yr 16

Diplomat (Yr 8)
  └─ War/Peace → shapes card pool for rest of run

Merchant (Yr 12)
  └─ Spend gold → buy cards directly

Prophet's Blessing (Yr 16)
  └─ Delayed effect from Yr 6

Tower Event (year ?)
  └─ +2 grid plots → more placement options

Final Battle (Yr 33)
  └─ Win = run complete / endless continues
  └─ Loss = all lives gone = run over
```
