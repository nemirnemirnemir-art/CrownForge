**Theme:** Movement, destruction, and expansion through chaos  
**Special mechanic:** Warlord is both Base AND Troop (moves on battlefield); Dragon's Den scales with destroyed plots; Migration Tome repositions cards

| # | Card | Type | Key Notes |
|---|------|------|-----------|
| 1 | Warlord | Base + Troop | Unique: fights AND is the base; mobile battlefield anchor for the run |
| 2 | Mob | Troop | Stone-throwing ranged troop; gains 1 unit for each adjacent unit added |
| 3 | Raptor | Troop | Mount unit that can be ridden by an adjacent troop |
| 4 | Frost | Enchantment | Freeze/slow tool that helps control enemy tempo |
| 5 | Dragon's Den | Tower | Summons single-strike dragons; +20% per destroyed plot (permanent) |
| 6 | Camp | Building | Adjacent plots gain +2%/+4%/+6% stats whenever any card is placed on an empty plot |
| 7 | Migration | Tome | Move a plot to an empty adjacent slot; moved card gets +10% DMG and HP |
