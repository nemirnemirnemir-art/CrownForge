## What This Is

Structured documentation of **9 Kings** (Steam Early Access, roguelike kingdom builder)
compiled from public sources: wiki.gg, Reddit, Steam community, review sites.

Used as: design reference, mechanic analysis, and template for original game development.

## Data Confidence Levels

All files use these markers:

- **[CONFIRMED]** — verified from wiki or multiple sources
- **[INFERRED]** — logical conclusion from confirmed data; no direct source
- **[UNCONFIRMED]** — reported but not verified; treat with caution
- **[MISSING]** — exact data not found in open sources; description only
