# Code Audit Reports

Automated analysis outputs for code quality tracking.

## Files

| File | Purpose |
|------|---------|
| `SUMMARY.md` | Executive summary with risk ratings |
| `BLOATED_SCRIPTS.md` | Large files needing refactoring |
| `FRAGILE_CODE.md` | High-risk code patterns |
| `SCRIPTS_MAP.md` | Script dependency mapping |
| `SCENES_MAP.md` | Scene structure mapping |
| `CODE_VS_SCENE.md` | Code-scene alignment |
| `DEPENDENCIES.md` | Dependency graph |
| `SIGNALS_MAP.md` | Signal connections |
| `RESOURCES_MAP.md` | Resource files mapping |
| `UNUSED_FILES.md` | Potentially unused files |
| `STRUCTURE.md` | Project structure analysis |

## Status

Snapshot as of audit date. Not auto-updated.

## Priority Legend

- 🔴 Critical - Requires immediate attention
- 🟡 High - Should address soon
- 🟢 Medium - Plan for later

## Usage

1. Start with `SUMMARY.md` for overview
2. Check `BLOATED_SCRIPTS.md` for refactoring targets
3. Review `FRAGILE_CODE.md` for risk areas
4. Use maps for navigation and understanding
