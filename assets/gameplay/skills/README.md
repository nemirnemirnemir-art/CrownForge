# Skills Textures

Place skill icon textures here (50x50 pixels).

- `skill1_autoclick.png` - Autoclick skill icon (50x50)
- `skill2_damage_boost.png` - Damage boost skill icon (50x50)
- `skill1_effect.png` - Visual effect for skill 1 autoclick (appears on enemy, optional)

After placing textures, assign them in the inspector:
- SkillsPanel -> skill1_texture
- SkillsPanel -> skill2_texture
