Научиться создавать пиксель-арт, используя строго ограниченную палитру (4 цвета), развивая навыки выбора гармоничных цветов и эффективного использования ограниченного набора цветов.

**Что вы освоите**:
- Выбор ограниченной палитры (4 цвета)
- Создание цветовой гармонии
- Эффективное использование каждого цвета
- Понимание важности ограничения цветов в пиксель-арте

## Референсные примеры

### Что делает пример хорошим

- **Ограничение**: Используется строго ограниченное количество цветов (4-6)
- **Гармония**: Цвета работают вместе, создают приятное сочетание
- **Эффективность**: Каждый цвет используется осознанно и эффективно
- **Читаемость**: Объект легко читается даже с ограниченной палитрой

### Принципы, которые применены

- **Ограниченная палитра**: Используется небольшое количество цветов (4-6)
- **Цветовая гармония**: Цвета подобраны так, чтобы работать вместе
- **Эффективное использование**: Каждый цвет имеет свою роль (фон, основной, тень, обводка)

### Как воспроизвести

1. Выберите 4 цвета, которые гармонично сочетаются
2. Определите роль каждого цвета (фон, основной, тень, обводка)
3. Используйте основной цвет для большей части объекта
4. Примените тень для создания объема
5. Добавьте обводку только если нужно для читаемости

## Типичные ошибки

### 1. Использование больше 4 цветов

**Симптомы**: В палитре объявлено 4 цвета, но фактически используется 5+ цветов

**Причина**: Создание промежуточных цветов, смешивание цветов, добавление "нужных" цветов

**Как исправить**: 
- Строго придерживайтесь 4 цветов
- Не создавайте промежуточные цвета
- Если нужен новый цвет - замените один из существующих

### 2. Цвета не гармонируют

**Симптомы**: Цвета конфликтуют, создают неприятное визуальное впечатление

**Причина**: Случайный выбор цветов без учета гармонии

**Как исправить**: 
- Используйте цветовой круг для выбора гармоничных цветов
- Выбирайте цвета из одной цветовой семьи или комплементарные
- Тестируйте сочетания перед использованием

### 3. Неэффективное использование цветов

**Симптомы**: Один цвет используется только для 1-2 пикселей, другой для всего объекта

**Причина**: Неправильное планирование роли каждого цвета

**Как исправить**: 
- Планируйте роль каждого цвета заранее
- Основной цвет должен использоваться для большей части объекта
- Тень и обводка - для акцентов и объема

### 4. Объект не читается

**Симптомы**: Объект теряется на фоне, не понятно что это

**Причина**: Недостаточный контраст между объектом и фоном

**Как исправить**: 
- Убедитесь, что объект контрастирует с фоном
- Используйте обводку для улучшения читаемости
- Увеличьте контраст между основным цветом и фоном
