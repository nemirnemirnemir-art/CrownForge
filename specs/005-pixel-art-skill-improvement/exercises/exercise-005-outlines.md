## Цель упражнения

Научиться правильно использовать обводку (outline) для улучшения читаемости спрайта. Обводка должна быть селективной — только там, где она нужна, и не должна перезаписывать важные детали.

**Что вы освоите**:
- Селективная обводка (только внешние края)
- Контраст обводки с фоном
- Избежание перезаписи деталей обводкой
- Правильный порядок наложения (обводка в последнюю очередь)

## Референсные примеры

### Что делает пример хорошим

- **Селективная обводка**: Обводка только на внешних краях, не внутри объекта
- **Контраст**: Обводка контрастирует с фоном, улучшая читаемость
- **Детали сохранены**: Важные детали (дырки, текстуры, акценты) не перезаписаны
- **Правильный порядок**: Обводка добавлена в последнюю очередь

### Принципы, которые применены

- **Селективность**: Обводка только там, где нужна (внешние края)
- **Контраст**: Обводка контрастирует с фоном
- **Порядок наложения**: Обводка в последнюю очередь, после всех деталей
- **Проверка перед перезаписью**: Детали не перезаписываются обводкой

### Как воспроизвести

1. Создайте базовую форму объекта
2. Добавьте блики, тени, детали
3. Добавьте обводку только на внешних краях
4. Проверьте, что детали не перезаписаны

## Типичные ошибки

### 1. Обводка перезаписывает детали

**Симптомы**: Детали (дырки, текстуры, акценты) перезаписаны обводкой

**Причина**: Обводка добавлена до деталей или без проверки

**Как исправить**: 
- Добавляйте обводку в последнюю очередь
- Проверяйте перед перезаписью: `if pixel_map[key] != detail_color`
- Пропускайте детали при добавлении обводки

### 2. Обводка внутри объекта

**Симптомы**: Обводка видна внутри объекта, не только на краях

**Причина**: Обводка добавлена ко всем пикселям, а не только к краям

**Как исправить**: 
- Проверяйте, является ли пиксель краем (нет соседей снаружи)
- Добавляйте обводку только к внешним краям
- Используйте проверку: `if not pixel_map.has(Vector2i(x, y - 1))` для краев

### 3. Обводка не контрастирует с фоном

**Симптомы**: Объект плохо читается на фоне

**Причина**: Цвет обводки слишком близок к цвету фона

**Как исправить**: 
- Используйте темную обводку для светлых объектов
- Используйте светлую обводку для темных объектов
- Проверяйте контраст перед финализацией

### 4. Обводка добавлена слишком рано

**Симптомы**: Детали перезаписаны обводкой, порядок наложения нарушен

**Причина**: Обводка добавлена до деталей

**Как исправить**: 
- Следуйте порядку: базовая форма → блики/тени → детали → обводка
- Добавляйте обводку в последнюю очередь
- Проверяйте порядок в коде
