# Финальный чеклист тестирования Hero Bar + Hero Card UI

## ✅ Базовый функционал (должно работать)

### 1. Отображение героев
- [ ] Панель HeroBar видна внизу экрана (компактная, не растягивается)
- [ ] Панель HeroCard видна справа внизу
- [ ] Герои появляются при найме (кнопка "LVL UP")
- [ ] Имена героев отображаются в HeroCard
- [ ] Иконки героев загружаются правильно (swordsman, archer, warrior_woman)

### 2. Пагинация
- [ ] Кнопки "<" и ">" работают
- [ ] При >20 героях можно переключать страницы
- [ ] Кнопки отключаются на первой/последней странице
- [ ] Панель компактная, нет пустого пространства справа

### 3. Выбор героев в отряд
- [ ] Клик по герою добавляет его в отряд (красная рамка)
- [ ] Можно выбрать до 6 героев
- [ ] 7-й герой не выбирается (молчаливый игнор)
- [ ] Повторный клик снимает выбор (рамка исчезает)
- [ ] HeroCard обновляется при выборе героя

### 4. HeroCard (карточка героя)
- [ ] Показывает имя героя
- [ ] Показывает HP (исправлен баг 15/11 → теперь корректно)
- [ ] Показывает Damage
- [ ] Показывает Level
- [ ] Показывает XP / XP_to_next
- [ ] Кнопки работают:
  - [ ] "+1 XP" - добавляет 1 XP
  - [ ] "+5 XP" - добавляет 5 XP (триггерит левелап)
  - [ ] "Kill" - помечает героя как мёртвого
  - [ ] "Info" - выводит отладочную информацию

### 5. XP и автолевелап
- [ ] Добавление XP обновляет HeroCard
- [ ] При достижении XP_to_next происходит левелап
- [ ] Level увеличивается
- [ ] HP или Damage увеличивается случайно
- [ ] XP сбрасывается (остаток сохраняется)

### 6. Мёртвые герои
- [ ] При нажатии "Kill" герой помечается как мёртвый
- [ ] В HeroBar показывается череп (или placeholder)
- [ ] Мёртвый герой не может быть выбран в отряд
- [ ] Клик по черепу удаляет героя (освобождает слот)
- [ ] HeroCard не меняется при клике на череп

### 7. Reset Progress
- [ ] При нажатии "Reset Progress" панель очищается
- [ ] Все слоты становятся пустыми
- [ ] HeroCard показывает "Выберите героя"
- [ ] После reset можно нанимать новых героев

### 8. Сохранение/загрузка
- [ ] Выбранные герои сохраняются (красные рамки)
- [ ] XP и уровень сохраняются
- [ ] Мёртвые герои сохраняются (череп)
- [ ] После перезапуска всё восстанавливается

### 9. Позиционирование
- [ ] Панель не сдвигается при каждом запуске
- [ ] Панель компактная, не растягивается
- [ ] Кнопки навигации рядом со слотами
